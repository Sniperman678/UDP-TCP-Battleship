-----------------------------------------------------------------Read Me-----------------------------------------------------------
Use IntelliJ
Ensure User has the most upto date version of java JDK18

Run Main File on both computer and ensure they are on the same network
-Read the prompts: 
	-User 1 start a new game by Entering "1"
	-User 2 joins/finds game by Entering "2"
	-Once Connection is established User 1 will fire at chosen location (FIRE:A7). this will display grids on both users screens
	-Once the users have completed the game it will end.

MAKE SURE:
-THE FIRE COMMAND IS TYPED CORRECTLY (FIRE:LOCATION)
